RbAda

Empresa: Artesanía Digital Spa.
Sitio Web: adispa.cl
Herramientas: Java, Kotlin, SQL, Android Studio, MQTT.

RbAda es una aplicacion para dispositivos móviles con sistema operativo Android que utiliza el protocolo MQTT para enviar y recibir mensajes de un servidor, tales como: lectura de sensores, alertas, estados de actuadores, imagenes, etc... Además posee 1 servicio propietario de notificaciones Push (el servicio permanece siempre activo con un consumo mínimo de bateria).

TrafficGPS

Empresa: TrafficGPS.
Sitio Web: TrafficGPS.cl
Herramientas: Java, Kotlin, SQL.

TrafficGPS es una aplicacion para dispositivos móviles con sistema operativo Android utilizada para controlar un cierto modelo de GPS para vehículos a través de SMS y llamadas telefónicas. TrafficGPS te da la opción de registrar múltiples vehículos y protegerlos con un usuario y contraseña.

Battle Car Royale

Empresa: Personal.
Sitio Web: Vídeo trailer
Herramientas: C++, OpenGl, Bullet Physics, OpenAL, GLSL.

BattleCarRoyale es un juego para sistemas Windows, linux y MacOS x, realizado principalmente como un proyecto para la Universidad Austral de Chile. El juego consta se batallas 1v1 a pantalla dividida entre una selección de 6 vehículos, tiene soporte para joysticks de Xbox 360. Algunas de las técnicas usadas son: Phong Shading, Shadow Mapping, Dynamic Lighting, Normal Mapping, entre otros.
